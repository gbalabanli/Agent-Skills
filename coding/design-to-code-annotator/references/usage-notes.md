# Annotation Tool Usage

1. Open `assets/annotation-tool/index.html` in a browser.
2. Upload one or more screen images.
3. Select a screen from the list.
4. Draw a rectangle over an interactive region.
5. Fill annotation fields and save.
6. Optionally set `Target Screen` to define a transition.
7. Export JSON and markdown.

Validation checklist:
- Required fields are populated for each annotation.
- `target_screen_id` is set where a transition exists.
- Exported markdown includes navigation flow.
