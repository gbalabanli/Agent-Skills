# Reference Style

Use a Wikipedia-like style: concise source title plus a direct URL.

## Entry Format

`[n] <Title>. <URL>`

`Relevance: <Short explanation of why this source matters for this document>`

Example:

`[1] Transformer (machine learning model). https://en.wikipedia.org/wiki/Transformer_(deep_learning_architecture)`

## Rules

- Keep one reference per line.
- Use direct page URLs where possible.
- Preserve source titles as provided unless obviously malformed.
- Add a short relevance note for each reference when available.
- Place the `References` section at the end of the document.
- Do not mix citation styles in a single output.
