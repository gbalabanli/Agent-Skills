# Edit Anchors

Prefer these insertion targets when editing an existing application form:

- `Supporting Information`
- `Why I am suitable`
- `Personal Statement`
- `Motivation`
- `Qualifications`
- `Additional Notes`
- the final free-text section before signature or declaration blocks

## Anchor Rules

- If the user names a section, use that exact section first.
- If the form has a free-text answer area, insert there before creating a new section.
- Avoid inserting new prose inside tables, checkbox grids, or signature blocks unless the user explicitly asks for that.
- If the form has no safe anchor, append a short section titled `Supporting Statement`.
- Keep inserted text proportional to the existing form. A one-page form usually needs one paragraph, not a full essay.
