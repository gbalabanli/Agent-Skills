# Postbridge Social Growth

Use this skill to coach execution of an organic short-form growth system.

## Workflow

1. Diagnose current stage.
- Determine whether user is starting new, stuck at low views, converting poorly, or ready to scale.
- Capture product, audience, and offer.
- Confirm platform mix and posting constraints.

2. Set up conversion foundation.
- Define the one conversion action per profile.
- Rewrite bio to state offer and next action clearly.
- Align profile, pinned content, and landing destination.

3. Run warmup protocol for new accounts.
- Follow warmup period before regular posting.
- Focus on niche relevance signals and manual engagement quality.
- Avoid scaling activity before baseline distribution appears.

4. Find content-market fit.
- Test one short-form concept daily using proven format adaptation.
- Track hook quality, retention, and comment velocity.
- Keep tests focused on audience pain or aspiration, not product pitch.

5. Layer trend adaptation.
- Add a small number of trend-based tests each week.
- Preserve proven structure while adapting to niche context.
- Reject trends that do not support conversion intent.

6. Optimize views-to-conversion funnel.
- Audit CTA placement in video, caption, and comments.
- Improve profile-to-landing continuity.
- Resolve friction between social promise and landing experience.

7. Scale only after proof.
- Require consistent posting history and repeatable winning formats.
- Require evidence of conversion from organic traffic.
- Increase output, account count, or platform scope in controlled steps.

## Output Rules

- Tailor advice to user stage instead of giving generic checklists.
- Keep recommendations testable with weekly metrics.
- Prefer repeatable systems over one-off viral bets.
- Do not recommend aggressive scaling before conversion proof.
