# Marketing Ideas

Use this skill to turn an open-ended brainstorm request into a focused experiment queue.

## Workflow

1. Capture constraints.
- Identify stage, revenue model, audience, and current traction.
- Confirm budget, team capacity, and timeline.
- Define the primary objective for this cycle.

2. Build idea universe.
- Generate ideas across SEO, content, social, lifecycle, partnerships, paid, and product-led growth.
- Pull candidate plays from [ideas-by-category.md](references/ideas-by-category.md).
- Remove channels clearly mismatched to audience or constraints.

3. Score and rank ideas.
- Score each idea with impact, confidence, effort, and time-to-signal.
- Flag dependencies and required skills.
- Rank into `now`, `next`, and `later` buckets.

4. Convert top ideas into briefs.
- For each top idea, define audience, promise, offer, and distribution path.
- Provide first 3 execution steps and required assets.
- Define success metric and failure threshold.

5. Assemble test plan.
- Recommend 3-5 ideas for the next sprint.
- Balance quick wins with compounding bets.
- Add weekly check-in rhythm and decision gates.

## Output Rules

- Return only prioritized ideas with execution detail.
- Avoid generic brainstorming dumps.
- Include clear stop rules to prevent sunk-cost drift.
- State assumptions when channel fit evidence is weak.
